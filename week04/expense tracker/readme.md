# Expense Tracker Lite

A simple interactive web app that allows users to:

- Add expenses
- Delete individual expenses
- Clear all expenses
- View live summary updates

## Features

- DOM manipulation using:
  - getElementById
  - addEventListener
  - createElement
  - appendChild
  - textContent
- Input validation
- Live total calculation
- Highest expense tracking

## How to Run

1. Open expense.html in your browser.
2. Add expense name and expense figure and run to see results
