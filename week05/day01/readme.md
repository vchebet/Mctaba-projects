# Sum Until Largest Prime (997)

## Description

This is a simple JavaScript web app that allows the user to enter numbers and keeps a running total. The program continues to add numbers until the **largest prime number under 1000** is entered, at which point it stops and displays the final sum.

The app demonstrates basic **JavaScript**, **DOM manipulation**, and **event handling**.

---

## Features

- Enter numbers using an input field.
- Live updating of the cumulative sum.
- Stops automatically when 997 is entered.
- Displays a message when the process is complete.
- Prevents invalid input (non-numbers).

---

## How to Use

1. Open `index.html` in a web browser.
2. Enter a number in the input field.
3. Click the **Add Number** button.
4. Repeat until you enter the largest primw number under 1000
5. The final sum will be displayed, and inputs will be disabled.
